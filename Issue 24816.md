Minor point, but it is quite a bit faster to do `tuple([gen])` than `tuple(gen)`:

```
sage: %timeit tuple([x for x in range(1000)])
10000 loops, best of 3: 28.1 µs per loop
sage: %timeit tuple(x for x in range(1000))
10000 loops, best of 3: 40.6 µs per loop
```
