In #29222 a memory leak when using p-adic polylogs was noticed, the code to reproduce is

```
for p in prime_range(100):
    K = Qp(p,prec=10)
    print(p)
    for t in K.teichmuller_system():
        if t == 1:      
            continue    
        l = t.polylog(2)
        print(p,l)
```
