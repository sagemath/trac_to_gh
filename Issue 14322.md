# Issue 14322: implicit_plot should be able to accept a list of functions

Issue created by migration from https://trac.sagemath.org/ticket/14526

Original creator: eviatarbach

Original creation time: 2013-05-03 22:22:44

Assignee: jason, was

Currently, `implicit_plot` only accepts one function at a time; `implicit_plot([x == y^2, x == e^y], (x, -1, 1), (y, -1, 1))` returns an error. It should be able to accept a list of functions, as does `plot` and `polar_plot`.
