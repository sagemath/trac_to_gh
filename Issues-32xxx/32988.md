<a id='comment:6'></a>That file was added in #16996, to support graphics output in the pre-jupyter era ipython notebook. It seems that as Sage went over to use jupyter rich-output facilities, the `GraphicsFile` class provided by the file became out of use. The function `graphics_from_save` also seems to have become the method of the same name of the display manager defined in `sage.repl.rich_output`.

Hence I am positive to deprecate this file.
