Currently the following fails, throwing a TypeError:

```
F = GF(3).algebraic_closure()
P.<x,y> = ProjectiveSpace(F, 1)
H = Hom(P, P)
f = H([x^2 + y^2, y^2])
f.reduce_base_field()
```

This ticket fixes this error.
