<a id='comment:7'></a>A good question for `@`isuruf then. `-nostdlib` is usually used to link C++ code completely statically. It should only be used in conjunction with `static`. If you try to link C++ statically without `-nostdlib` various standard library will still be dynamically linked. It is a nuclear option to get statically linked C++ code.
