```
sage: h(t) = sqrt(abs(-2*I*cos(t)*sin(t) + 2*sin(t)^2))
sage: h.simplify_full()
```

```
TypeError: ECL says: Error executing code in Maxima: sign: argument cannot be imaginary; found %i
```
