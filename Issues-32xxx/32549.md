MPIR, a fork of GMP, has not been updated upstream. The latest release, 3.0.0, is from 2017-03-01 (https://mpir.org/downloads.html). 

Sage currently provides an SPKG for MPIR as a configurable alternative to GMP. We should remove this option and package, as previously suggested in [#29620 comment:10](https://github.com/sagemath/sage/issues/29620#comment:10)

 

Depends on #30350
Depends on #31163
