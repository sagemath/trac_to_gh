```
sage: x = polygen(QQ)
sage: f = x^3 - x^2 + 2*x + 8
sage: K.<a> = NumberField(f)
sage: K.completely_split_primes(80)
[59, 73, 79]
sage: len((2*K).factor())
3
```

Since `len((2*K).factor()) == 3` we should expect 2
to show up in the list of completely split primes.
