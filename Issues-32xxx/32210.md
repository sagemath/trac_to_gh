The right multiplication of a Ore polynomial by 0 outputs a result with a wrong degree:

```
sage: k = GF(5)
sage: A.<t> = k[]
sage: d = A.derivation()
sage: S.<X> = A['X', d]                                                                                                                                                                                   
sage: f = 0*X
sage: f.degree()
1
```
