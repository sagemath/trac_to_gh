<a id='comment:5'></a>I wonder if we should really be modifying `$LIBS` at all.  Each time we do an `AC_CHECK_LIB` or similar, that passes, that lib is getting added to `$LIBS`, and thus every subsequent library check is over-linking (with all the `-l` flags added via previous `AC_CHECK_LIB` calls).

It hasn't been a problem yet that I know of, and it might even be useful.  Just thought I would mention it.
