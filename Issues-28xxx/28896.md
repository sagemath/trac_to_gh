There is a few methods that are implemented exactly the same way in dense and sparse graph:

- add_arc
- has_arc
- del_all_arcs
- out_neighbors
- in_neighbors

All they do is wrap the corresponding unsafe methods. So it makes sense to move that code all to `CGraph`.
