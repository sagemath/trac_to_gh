Do you think this need a patch ?

IMHO it's not necessary. You can call `__mpq__` explicitly if needed :

```
sage: from gmpy2 import mpq
sage: q = 5/3
sage: q.__mpq__() == mpq(5,3)
True
sage: mpq(q) == mpq(5,3)
True
```
