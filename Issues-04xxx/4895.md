The following behavior is clearly bad:

```
sage: [p for p in Permutations(4,avoiding=[2,3,1]) if p.has_pattern([2,3,1])]               
[[2, 3, 1, 4], [4, 2, 3, 1]]
```

Similar behavior occurs when avoiding [1,3,2], [2,1,3], and [3,1,2].
