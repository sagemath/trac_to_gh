This now works:

```

[ -8   2   0]
[  0   1  -1]
[  2   1 -95]
sage: As = singular(A); As
-8     2     0
 0     1    -1
 2     1   -95
sage: As._sage_()
[ -8   2   0]
[  0   1  -1]
[  2   1 -95]
```
