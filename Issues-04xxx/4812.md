```
Hello all,

I'm trying to run the following code:

s     = 7
s2    = 2^s
P.<x> = GF(2)[]
M     = matrix(parent(x),s2)
for i in range(s2):
  p  = (1+x)^i
  pc = p.coeffs()
  a  = pc.count(1)
  for j in range(a):
    idx        = pc.index(1)
    M[i,idx+j] = pc.pop(idx)
matrixprogram = matrix_plot(M,cmap='Greys')

...but with 3.2.1, it complains:
```
