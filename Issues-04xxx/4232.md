Try the following in a sage that contains the patch at #249

```
a=[(i,randint(0,100)) for i in range(3000)]                  
f=open("mytest.sage",'w')                  
f.write("a=[\n")                           
f.writelines(["%s,\n"%str(i) for i in a])  
f.write("(0,0)]")                          
f.close()
load mytest.sage            
```
