```
sage: p = MixedIntegerLinearProgram()                                                                                                                                              
sage: x,y = p['x'], p['y']                                                                                                                                                         
sage: p.set_min(x,None)                                                                                                                                                            
sage: p.set_min(y,None)                                                                                                                                                            
sage: p.add_constraint( x <= 1 )

sage: p.add_constraint( y <= 1 )                                                                                                                                                   
sage: p.add_constraint( y >= -1 )
```
