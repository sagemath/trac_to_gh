Keywords: sd87

In number fields, there is a method for accessing the polynomial underlying an element:

```
sage: S.<x> = QQ[]
sage: K.<a> = NumberField(x^3 -x + 27)
sage: a.polynomial()
x
```

There should be a similar method for `p`-adic extensions.
