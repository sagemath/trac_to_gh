CC:  @nbruin

In examples like this:

```
sage: f(x)=matrix()
sage: bool(f(x)-f(x)==0)
...
TypeError: mutable matrices are unhashable
sage: f(x)*1
...
TypeError: mutable matrices are unhashable
```

a Pynac bug manifests, see https://github.com/pynac/pynac/issues/34

This ticket should provide doctests, it depends on pynac-0.4.2/0.3.9.2.
