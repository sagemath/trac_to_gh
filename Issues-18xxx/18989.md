The following behaviour is a bug:

```
sage: A = Automaton([(0, 0, 0)])
sage: B = Automaton([(0, 0, 1)])
sage: A.input_alphabet
[0]
sage: B.input_alphabet
[1]
sage: (A + B).input_alphabet
[0]
```
The resulting input alphabet should be `[0, 1]`.

Depends on #18557
