In cython code, replace

```
cdef type t = x.__class__
```
by

```
cdef type t = type(x)
```
For Cython extension types, we know that `type(x)` is `x.__class__`, so this code should be equivalent, but `type(x)` is a lot faster because Cython optimizes it.
