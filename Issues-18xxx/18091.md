With

```
sage: f(x) = floor(x)
sage: g(x) = ceil(x)
sage: h(x) = factorial(x)
```

```
sage: f(pi)._numerical_approx()
sage: g(pi)._numerical_approx()
sage: h(5)._numerical_approx()
```
will result in 

```

-> 4890             res = py_object_from_numeric(x._gobj)
   4891         elif  is_a_constant(x._gobj):
   4892             res = x.pyobject()
```
