The following should be made to work:

```
sage: A.<a,b,c,d,e,f> = FreeAlgebra(QQ, implementation='letterplace')
sage: p = a*b*a*c*d*a*f
sage: p[3:-1]
c*d*a
```
So, a `__getitem__` method should be implemented for monomials of free algebras.

