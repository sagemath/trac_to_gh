```
f = (2*(x - floor(x))^3 - 3*(x - floor(x))^2 + x - floor(x))*(10*x^3/(x^2 + 1)^6 - 3*x/(x^2 + 1)^5)
```

```
ans_sage = integrate(f, x, 1, +Infinity)

```

```
ans_sage.n()
```

seems like ans_sage is wrong

```
from scipy import integrate

```

```
ans_scipy[0]
```
