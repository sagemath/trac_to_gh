See #13313 and the comments [at the end of](https://github.com/sagemath/sage/issues/16455#comment:27) #16455.  Basically, a patch was removed that allows pyzmq to build on old (32-bit) Macs - the problem is that pyzmq assumes one has the Mac version of gcc which supports the `-arch` flag, which we do not since we build our own vanilla gcc.
