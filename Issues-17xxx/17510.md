See #13313 and the comments [comment:27:ticket:16455 at the end of] #16455.  Basically, a patch was removed that allows pyzmq to build on old (32-bit) Macs - the problem is that pyzmq assumes one has the Mac version of gcc which supports the `-arch` flag, which we do not since we build our own vanilla gcc.
