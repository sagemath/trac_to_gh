There HAS to be a better way than the current one.

```
    n = abs(int(n))
    X = sorted(set([ZZ((a*a)%n) for a in range(n//2+1)]))
    return X
```
Perhaps using bitsets instead of `set()` and doing the multiplication in `ZZ` instead of using Python `int`s.
