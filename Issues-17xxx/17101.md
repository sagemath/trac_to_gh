Current erroneous behaviour:

```
sage: v = vector([RIF(-1, 1)], sparse=True)
sage: v.is_zero()
True
sage: v
(0)
```

The reason is the check `x[i] != 0` in `__init__`:

```
sage: RIF(-1, 1) != 0
False
sage: RIF(-1, 1) == 0
False
sage: bool(RIF(-1, 1))
True
sage: bool(RIF(0, 0))
False
```
