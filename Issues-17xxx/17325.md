The following code does not return the right answer

```
PS.<x,y> = ProjectiveSpace(QQ,1)
Q=PS([3,2])
Q.scale_by(2/3)
print Q
Q.clear_denominators()
Q
```

the issue appears that it is not examining the last coordinate, it really needs dimension_relative+1.
