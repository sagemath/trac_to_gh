Expression normalization only works with standard polynomials:

```
sage: (x^2 + x^(-2) - 2).normalize()
(x^4 - 2*x^2 + 1)/x^2

vs

sage: (x^(2*pi) + x^(-2*pi) - 2).normalize()
x^(2*pi) + x^(-2*pi) - 2
```


This is necessary for being able to factor all symbolic expressions.
