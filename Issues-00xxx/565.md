<a id='comment:2'></a>This is a pretty shocking before and after example!

BEFORE

```
age: get_memory_usage()
372.22265625
sage: m = random_matrix(ZZ, 300, sparse=True)
sage: get_memory_usage()
376.92578125
sage: for i in range(300):
....:     for j in range(300):
....:         m[i,j] = 0
....:

sage: get_memory_usage()
786.4765625
```


A 300 MB memleak!!!

AFTER:

```
sage: get_memory_usage()

sage: for i in range(300):
....:     for j in range(300):
....:         m[i,j] = 0
```
