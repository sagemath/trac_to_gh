A file with this content (w/o the "==="'s) shows the problem:

```
def Foo(x)
    x = 1
```

After load/attaching this file, typing "??Foo" prints only the 'def' line.

If the file looks like this (i.e., with an 'extra' blank line):

```
def Foo(x)
    x = 1

```

then '??' works properly.
