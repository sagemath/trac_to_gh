SAGE should support iterating over sparse multivariate polynomials like this:

```
sage: P.<x,y,z> = PolynomialRing(QQ,3)
sage: f= 3*x^3*y + 16*x + 7
sage: for c,m in f:
....:     print c,m
....:
3, x^3*y
16, x
7,1
```
