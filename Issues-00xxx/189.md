This input to a notebook cell:

```
if gap('1+1 = 3'): 
    print "it is true"
else:
    print "it is false"
///
it is false
```

Click on "text" and get this out, which is wrong:

```
sage: if gap('1+1 = 3'): 
...    print "it is true"
...
sage: else:
...    print "it is false"
it is false
```

It should be:

```
            sage: if gap('1+1 = 3'): 
            ...    print "it is true"
            ... else:
            ...    print "it is false"
            it is false
```
