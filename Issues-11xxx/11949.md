<a id='comment:5'></a>I tried this patch in 4.8.alpha0 for many examples, for which I previously used the Coin-cbc solver or the standalone versions of scip and gurobi via the mps-export of the linear program. It seems to work fine with Gurobi versions 4.5 and 4.6. The docs are updated, and the thematic tutorial on linear programming explains well how to make gurobi visible to sage.

It's great that this powerful solver (for which one can get a free academic license) can now be used from inside Sage.
