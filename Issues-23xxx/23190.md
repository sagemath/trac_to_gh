This should return `True`

```
sage: R.<x> = ZZ[]
sage: S.<x> = QQ[]
sage: S.quo(x^2 + 1).coerce_map_from(R.quo(x^2 + 1)).is_injective()
```

Generally, if `R→S` is injective/surjective then the quotient is.
