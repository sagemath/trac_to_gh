Not all ring elements provide `inverse_of_unit()`

```
sage: R.<x> = ZZ[]
sage: S = R.quo(x^2 + x + 1)
sage: S(1).inverse_of_unit()
NotImplementedError
```

But we could always try to invert and see if the result is in the same parent.
