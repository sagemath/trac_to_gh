```
sage: P = PermutationGroup(['(1,2)','(1,3)'])
sage: P.cardinality()
6
sage: type(P.cardinality())
<type 'int'>
```
This prevents using the `.is_one()` method or any other `Integer` specific method.

The problem comes from the shortcuts implemented in the `_order` method.
