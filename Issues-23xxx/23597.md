```
+.. [BIANCO] L. Bianco, P. Dell‘Olmo, S. Giordani
```
you need to add a backslash before the first initial.

```
+.. [BIANCO] \L. Bianco, P. Dell‘Olmo, S. Giordani
```
for obscure sphinx technical reasons. Otherwise Sphinx thinks that this is an letter-numbered item.
