<div id="comment:5" align="right">Comment 5</div>

Don't put Python code inside `sig_on()`/`sig_off()` blocks.

What you are doing is essentially

```
sig_on()
try:
    initialize()
    res = c_lib_call()
    return process(res)
finally:
    sig_off
```
Instead, it should be

```
initialize()
sig_on()
res = c_lib_call()
sig_off()
return process(res)
```
