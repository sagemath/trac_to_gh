```

[2 1 0]
```

<a id='comment:9'></a>Note that in numpy, there is method to do reflection that is called flip. What I am trying to achieve with this ticket corresponds to

```
sage: import numpy as np
sage: a = numpy.array([[0,1],[2,3],[4,5]])
sage: np.flip(np.flip(a,0), 1)
array([[5, 4],
       [3, 2],
       [1, 0]])
```
And numpy provides shortcuts `flipud` (up/down) and `fliplr` (left/right) for flipping with respect to axes 0 and 1.
