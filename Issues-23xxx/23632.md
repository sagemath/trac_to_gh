```
sage: P.<x,y> = InfinitePolynomialRing(QQ)
sage: R = PolynomialRing(ZZ, ['x_3', 'y_2'])
sage: P.has_coerce_map_from(R)
False
```
but should be `True`. Note that conversion works already:

```
sage: P(R.0 + R.1)
x_3 + y_2
```
