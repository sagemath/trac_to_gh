```
sage: K.<s, t> = QQ.extension([x^2-2, x^2-3])
sage: R.<y, z> = K[]
sage: D.<d, e> = R[]
sage: R(D(y))
```
fails with `TypeError: not a constant polynomial`

```
sage: K.<s, t> = QQ.extension([x^2-2, x^2-3])
sage: R = PolynomialRing(K, 'xG_0, xG_1, xH_0')
sage: D.<d, e> = R[]
sage: R(D(R.0))
```
fails with `TypeError: unable to convert 'xG_0' to a rational`
