The root finder in `RIF` sometimes returns roots which have too much precision:

```
sage: R.<x> = QQ[]
sage: parent((x-1).roots(RIF)[0][0])
Real Interval Field with 106 bits of precision
```
The answer should have 53 bits of precision.
