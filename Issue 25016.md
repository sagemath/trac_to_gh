Keywords: Singular

There doesn't seem to be any way to create nested lists using Sage's Singular interface:


```
sage: b = singular.list([1, 2, [3, 4]])
sage: b
[1]:
   1
[2]:
   2
[3]:
   4*gen(2)+3*gen(1)
sage: b = singular.list([1, 2, singular.list([3, 4])])
```


The last command just hangs indefinitely.
