<a id='comment:2'>**Comment 2:**</a>
I'm surprised the exponential integral is not efficiently implemented elsewhere already. Note, however, that you could compute this more efficiently by keeping track of the single value (sgn*alpha_pow*denom) which changes by a constant factor of -1/(alpha * t) each iteration (if this is a bottleneck).
