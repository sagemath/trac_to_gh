There is a sign error in the Steenrod multiplication routine at odd primes: with

  ```
    A=SteenrodAlgebra(3) 
    Q=A.Q_exp
    P=A.P 
    print (Q(0,1)*P(3))*Q(1,1) 
    print Q(0,1)*(P(3)*Q(1,1))
  ```

one gets

  ```
    Q_0 Q_1 Q_2 
    2 Q_0 Q_1 Q_2
  ```

Only the 2nd answer is correct.
