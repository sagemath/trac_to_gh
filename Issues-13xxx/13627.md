```
sage: R = Zp(3)
sage: R.one().xgcd(R.zero())
AttributeError: 'sage.rings.padics.padic_capped_relative_element.pAdicCappedRelativeElement' object has no attribute 'xgcd'
```

The attached patch implements it.
