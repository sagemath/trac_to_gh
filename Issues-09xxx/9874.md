We should define a new symbolic function for the derivative of `ceil` or `floor`. 

In Maple:

```
> diff(floor(x),x);
                                  floor(1, x)
> diff(floor(x),x,x);
                                  floor(1, x)
> diff(ceil(x),x,x); 
                                  floor(1, x)
> eval(diff(ceil(x),x,x),x=1.5);
                                       0
> eval(diff(ceil(x),x,x),x=0);  
Error, (in floor) floor is not differentiable at integers
```
