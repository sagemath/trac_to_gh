<a id='comment:1'></a>It seems the `sage -testall` and `sage -tnew` operators invoke `SAGE_LOCAL/bin/sage-maketest` and `sage-test-new`, respectively.  To run the tests, both scripts use `sage -t`, which calls the serial doctest runner `sage-test`.  Moreover, `sage-test-new` does not pass along command-line arguments.
