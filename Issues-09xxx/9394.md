With the new pynac package at #9901, we have:

```
sage: var('n')
n
sage: t = 2*I*n*pi*x - 2*I*n*pi
sage: latex(t)
2 i \, \pi n x - 2 i \, \pi n
```

[attachment: trac_9394-leading_minus.patch](https://github.com/sagemath/sage/files/ticket9394/trac_9394-leading_minus.patch.gz) contains the doctest changes. The fixes in the printing of rational functions (for #9834) are also included in this patch.

The pynac package includes patches for #9834, #9878, #9879, #9881, #9900 as well as this ticket. See the ticket description of #9901 for the list (and order) of patches associated to the new version.
