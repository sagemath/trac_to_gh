The is a bug in the matrix constructor. If the entries are given by a method, the output matrix is always square of dimension ncols

```
sage: matrix(QQ, 1, 3, lambda x,y: x)
[0 0 0]
[1 1 1]
[2 2 2]
sage: sage: matrix(QQ, 3, 1, lambda x,y: x)
[0]
```
