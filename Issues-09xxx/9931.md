Currently, points on elliptic curves cannot automatically be converted to PARI:

```
sage: E = EllipticCurve([0,0,0,3,0])
sage: P = E.point([1,2]); P
(1 : 2 : 1)
sage: pari(P)
Traceback (most recent call last):
...
RuntimeError: evaluating PARI string
```
