1. You cannot just put docstrings in arbitrary places. So in `src/sage/combinat/lyndon_word.py`, you should replace

```
"""
docstring

code

doctests

```
by

```
"""
docstring

doctests

code
```

2. The indentation of the docstring is wrong:

```
TESTS::

```
should become

```
TESTS::

    sage: ...
```

3. Instead of testing `import *`, I would prefer to see a more concrete import (say `LyndonWord`)
