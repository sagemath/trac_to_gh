This is very bad:

```
sage: R.<x> = LaurentSeriesRing(ZZ)
sage: f = x^-2 + O(x^0); f
x^-2 + O(1)
sage: g = x^-1 + O(x^0); g
x^-1 + O(1)
sage: f == g
True
sage: f-g
x^-2 - x^-1 + O(1)
```

Also this is bad:

```
sage: R.<x> = LaurentSeriesRing(ZZ)
sage: x^(10^10) > 0
...
MemoryError: 
```
