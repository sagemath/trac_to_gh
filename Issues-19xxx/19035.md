With pynac-0.5 this will no longer give wrong results:

```
sage: assume(x,'integer')
sage: x.is_integer()
False

sage: assume(x>0)
sage: x.is_positive()
False

sage: f=function('f')(x)
sage: assume(f>0)
sage: f.is_positive()
False
```
