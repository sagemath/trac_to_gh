```
@coerce_binop
def gcd
```

With that change:

```
sage: R.<x> = PolynomialRing(ZZ,sparse=True)
sage: S.<y> = R[]

1
sage: _.parent() is S
True
sage: (6*x).gcd(9)
3
sage: _.parent() is R
True
```

Is there any reason I am unaware to perform manually the coercion?
