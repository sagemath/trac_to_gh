The algorithm for square roots mod 2<sup>n</sup> is the most naive one

```
for x in candidates:
    if x**2 == a:
        return x
```

(see lines 3722-3724 of `sage/finite_rings/integer_mod.pyx`). We implement a straightforward algorithm that runs in logarithmic time.
