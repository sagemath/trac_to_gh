How to install the wrapper

- install the spkg:

```
sage -f http://trac.sagemath.org/sage_trac/raw-attachment/ticket/12697/string_process-1.0.spkg
```
- install the patch:

```
cd $SAGE_ROOT/devel/sage
hg qinit
hg qimport http://trac.sagemath.org/sage_trac/raw-attachment/ticket/12697/trac_12697-string_process-fh.patch
hg push
sage -b
```
The wrapper should be installed.
