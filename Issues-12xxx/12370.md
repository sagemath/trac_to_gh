
<a id='comment:1'></a>Changed `is_singular` method to use rank instead of determinant, which is much faster for generic matrices (particularly over ZZ). Fixed also an inconsistency in the error raised by `determinant` for nonsquare matrices, as it returns different error messages over the rationals.

Change is fairly minimal, and tests pass on my machine (Mac OS 10.6.8). Ready for review.
