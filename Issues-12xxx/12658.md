This patch makes the print representation of lists of elements of SL2Z more legible.
Before:

```
sage: SL2Z.gens()
([ 0 -1]
[ 1  0], [1 1]
[0 1])
```
After:

```
sage: SL2Z.gens()
(
[ 0 -1]  [1 1]
[ 1  0], [0 1]
)
```

Depends on #11709
