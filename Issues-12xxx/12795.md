```
sage: G = F(u=b)
sage: H = G.simplify_trig()
sage: R = RealField(1000)
sage: L = [(j,H(R(j))) for j in srange(-1,.2,.001)]
sage: M = [(a,R(b)) for a,b in L] # should have done this in one step, but oh well
sage: points(M)+plot(G,(x,-1,.2),color='red')
```
