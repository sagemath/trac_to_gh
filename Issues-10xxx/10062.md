<a id='comment:1'></a>This was caused by the problem reported at #10064. The `arctan2()` automatic evaluation function checks if the second argument is positive, line 1062 of `inifcns_trans.cpp` in pynac:

```

		if (x.info(info_flags::positive))
			return _ex0;
```
