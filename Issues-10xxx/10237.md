Without the patch:

```
sage: Cone([], lattice=ZZ^2).polyhedron()
The empty polyhedron in QQ^0.
```

With the patch:

```
sage: Cone([], lattice=ZZ^2).polyhedron()
A 0-dimensional polyhedron in QQ^2 defined as the convex hull of 1 vertex.
```
