Patch applies cleanly, and doctest passes both before and after application.

Running

```
sage: R = BooleanPolynomialRing(256,'x')
sage: LR = [R.gen(i) for i in range(256)]
```

```
sage: R = BooleanPolynomialRing(256,'x')
```

```
R(S.gen(8)
```

in this situation.
