Keywords: matrix polynomial ring

As of 4.6.0, the following code hangs:

```
    sage: MS = MatrixSpace(ZZ, 2, 2)
    sage: R.<x> = MS[]
```
whereas 

```
    sage: MS = MatrixSpace(ZZ, 2, 2)
    sage: R.<x> = PolynomialRing(MS)
```
works.
