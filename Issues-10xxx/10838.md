```
        sage: from sage.quadratic_forms import isoautom
        sage: s = """#1
        ... 3x0
        ... 2
        ... 0 3
        ... 0 0 5
        ... 3x0
        ... 5
        ... 0 3
        ... 0 0 1
        ... """
        sage: x = isoautom.call_main_isom(['isom'], s)
        sage: x
        'The lattices are not isomorphic !\nDifferent numbers of short vectors (5|2)\n'

        ... 3x0
        ... 1
```
