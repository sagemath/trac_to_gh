Before:

```
sage: s1 = RealSet(0,2)
sage: s1.union(1,3)
(0, 3)
```
After, `s1.union(1,3)` is no longer allowed. 
Must be written as
`sage: s1.union((1,3))` or `s1.union(RealSet(1,3))` or `s1.union(RealSet.open(1,3))`, etc.

See [#32181 comment:58](https://github.com/sagemath/sage/issues/32181#comment:58)

Depends on #32181
