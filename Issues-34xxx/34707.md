This is a really bad associativity failure:

```
sage: r = ZZ['r'].fraction_field().gen()
sage: Q = QuadraticForm(r.parent(), 2, [0,r,0])
sage: C.<p,q> = CliffordAlgebra(Q)
sage: q * (p * q)
r*q
sage: (q * p) * q
-r*p*q + r^2
```
