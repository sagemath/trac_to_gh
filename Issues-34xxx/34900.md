As reported in [#34537 comment:145](https://github.com/sagemath/sage/issues/34537#comment:145), the following code fails with a `PariError`:

```sage
sage: GF(41^16).inject_variables()
Defining z16
sage: E = EllipticCurve([z16, 1])
sage: E.gens()
# ...

```
