We currently doctest

```
def quo_rem(self, right_r):
...
            sage: (t^-2 + 3 + t).quo_rem(t^-4 + t)
            (0, 1 + 3*t^2 + t^3)
```

I think we want 

```
sage: q, r = num.quo_rem(den)
sage: num == den * q + r
True
```
