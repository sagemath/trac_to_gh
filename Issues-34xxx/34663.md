```
+    subprocess.check_call(f'gcc -o {runmefile} ' + os.path.join(tempdir, '*.c ') +
```

```
+                          os.path.join('$SAGE_LOCAL', 'lib', 'libTIDES.a') + ' $LDFLAGS ' +
+                          os.path.join('-L$SAGE_LOCAL', 'lib ') + '-lmpfr -lgmp -lm  -O2 -w ' +
```
