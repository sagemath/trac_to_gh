```
sage: T = Graph()                                              
sage: T.add_vertices([1,(2,3)])                                
sage: list(T.orientations())                                   
---------------------------------------------------------------------------
TypeError                                 Traceback (most recent call last)

   3775 
   3776         if not self.size():
-> 3777             D = DiGraph(data=[self.vertices(), []],
   3778                         format='vertices_and_edges',
   3779                         name=name,
```
