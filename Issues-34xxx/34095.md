The following call fails:

```
P.<x,y> = ProjectiveSpace(ZZ, 1)
f = DynamicalSystem_projective([x^2 - y^2, y^2])
f.preperiodic_points(1, 1)
```

as dimension is called on a subscheme defined over a ring. We fix this error.
