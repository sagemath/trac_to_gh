Large exponents overflow to negative in polydict ring:


```
sage: from sage.rings.polynomial.multi_polynomial_ring import \
...       MPolynomialRing_polydict
sage: ring = MPolynomialRing_polydict(ZZ, 3, ['a','b','c'], "lex")
sage: a = ring.gens()[0]

sage: a^(2^31-1)
a^2147483647

sage: a^(2^31)
a^-2147483648

sage: a^(2^32)
1

```
