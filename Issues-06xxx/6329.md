It seems like the correct fix would be to always put the denominator in the `_magma_init_` method.

```
        s = self.numerator()._magma_init_(magma)
        s += '/' + self.denominator()._magma_init_(magma)
        return s
```

instead of 

```
        s = self.numerator()._magma_init_(magma)
        if not self.is_integral():
            s += '/' + self.denominator()._magma_init_(magma)
        return s

```
