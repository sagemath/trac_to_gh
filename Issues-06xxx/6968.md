for the record:

before:

```
sage: m=identity_matrix(1000,sparse=True)
sage: v=vector([1]*1000,sparse=True)
sage: time p = v*m
CPU times: user 2.26 s, sys: 0.00 s, total: 2.26 s

```

after:

```
sage: m=identity_matrix(1000,sparse=True)
```
