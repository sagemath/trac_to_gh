<a id='comment:8'></a>robharron points out that it would be more robust to create the Hecke algebra as an abstract ring (represented internally as a quotient of a polynomial ring over Z) plus maps back and forth (e.g., given N, turn the N-th Hecke operator into an abstract ring element). Then one could deal with ideals on the ring side, which would give access to all existing commutative algebra functionality.
