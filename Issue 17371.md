# Issue 17371: Allow use of principal value of integral if it's right

Issue created by migration from Trac.

Original creator: kcrisman

Original creation time: 2015-01-09 02:37:38


```

(%i1) integral(sec(x), x, -%pi/4, %pi/4);
                                              %pi  %pi
(%o1)                   integral(sec(x), x, - ---, ---)
                                               4    4
(%i2) integrate(sec(x), x, -%pi/4, %pi/4);
Principal Value
                         sqrt(2) + 2          sqrt(2) - 2
(%o2)                log(-----------) - log(- -----------)
                              2                    2

```

but Sage has

```
sage: integral(sec(x), x, -pi/4, pi/4)
ValueError: Integral is divergent
```

because of the PV.  See also #13511.
