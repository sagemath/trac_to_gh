<a id='comment:6'></a>The Python 3 version of the `pprint` module sorts objects safely e.g. like how it sorts dicts: https://github.com/python/cpython/blob/3.7/Lib/pprint.py#L180 by using a [_safe_key](https://github.com/python/cpython/blob/3.7/Lib/pprint.py#L72) class which basically wraps objects in such a way that restores Python 2-style comparison.

What we really need is a version of this in Sage.  It's too bad the one in Python is just an internal implementation detail in a relatively obscure module.  But it's easy enough to replicate.  We can also provide a `safe_sorted` which ensures each sort key is automatically wrapped.
