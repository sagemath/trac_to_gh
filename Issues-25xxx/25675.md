I think the issue with [comment:5](#comment:5) is that the cited instances of `face_lattice` don't currently appear in the reference manual, so there is nothing for them to link to. ```diff
diff --git a/src/doc/en/reference/discrete_geometry/index.rst b/src/doc/en/reference/discrete_geometry/index.rst
