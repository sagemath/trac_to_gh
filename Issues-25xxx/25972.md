Currently, assumptions are not always taken into account by `solve`.

```
sage: assume(x > 0)

# incorrectly gives [x < -1]
sage: solve(x^2 > 1, x)
[[x < -1], [x > 1]]

# correctly restricts
sage: solve(x^2 == 1, x)
[x == 1]
```

**Keywords:** inequality, solve
