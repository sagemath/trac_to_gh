Metaticket to track tasks and sub-tickets related to RSK refactoring project in GSoC'19.

Note: RSK stands for the Robinson-Schensted-Knuth correspondence or algorithm,
a combinatorial bijection between certain matrices with integer entries and
certain pairs of Young tableaux, see
[Wikipedia: RSK](https://en.wikipedia.org/wiki/Robinson–Schensted–Knuth_correspondence).

Tickets:

- #27852: Refactor structure of RSK
- #28058: Implement dualRSK
- #25070: Dual RSK and coRSK algorithms
- #24894: Add superRSK algorithm
- #28222: Implement shiftedRSK algorithm
- #28228: Implement semistandard and standard Supertableaux
- #28229: Generalized shifted prime tableau
