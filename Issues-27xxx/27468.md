```
sage: G = GL(2,5)
sage: g = G( matrix([[1,0],[0,4]]))
sage: H = G.subgroup([g])
sage: g in H
False
```

```
sage: g.gap() in H.gap()
True
```
The underlying reason should be

```
sage: G.has_coerce_map_from(H)
False
```
