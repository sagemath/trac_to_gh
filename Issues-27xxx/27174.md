The absolute value of a power is computed incorrectly when the base is a symbolic real number and the exponent a non-real complex number:

```
sage: abs(2^I)
2^I
```

When the exponent is a variable, the result is however correct:

```
sage: abs(2^x)
2^real_part(x)
```

CC:  @rwst

Status: new
