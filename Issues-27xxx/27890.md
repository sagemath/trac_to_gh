Pickling does not work correctly for codes:

```
sage: C = codes.HammingCode(GF(2), 3)
sage: C.decoders_available()
['InformationSet', 'NearestNeighbor', 'Syndrome']
sage: save(C, 'obj')
```

but in a fresh sage run,

```
sage: D = load('obj')
sage: D.decoders_available()
[]
```
