The behavior of `show` could be improved using `assume`.

Start with:

```
x = var('x')
f(x) = 1 - x
assume(x < 1, x > 0)
```
The current behavior is that `show(f)` will output `-x + 1`,
whereas a human mathematician would always write this as `1 - x`.
