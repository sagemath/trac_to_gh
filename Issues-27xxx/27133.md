Small speed up by avoiding duplicated operations.

before

```
sage: %timeit b = digraphs.ButterflyGraph(10)
10 loops, best of 3: 145 ms per loop
```
after

```
sage: %timeit b = digraphs.ButterflyGraph(10)
10 loops, best of 3: 123 ms per loop
```
We also do a pep8 cleaning.
