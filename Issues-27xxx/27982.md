This ticket makes the following work:

```
P.<x, y> = ProjectiveSpace(GF(37), 1)
E = EllipticCurve([1, 1])
f = P.Lattes_map(E, 2)
```
It failed up to now since `E` is defined over `QQ` and `P` is defined over `GF(37)`.
We should try to coerce to the base ring of projective space.
