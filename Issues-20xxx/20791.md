The following should return `True`:

```
sage: K.<a> = NumberField(x^2 + 1)
sage: K.extension(x^2 - 2, 'b') is K.extension(x^2 - 2, 'b')
False
```
This happens because `NumberField.extension()` does not use the `NumberField` factory.
