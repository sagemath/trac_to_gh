<a id='comment:4'></a>* To make the code Python3-compatible, you should replace
  {{{#!python
          return "\\overline{%s}" % self.base_ring()._latex_()

  by 
  {{{#!python
          return "\\overline``".format(self.base_ring()._latex_())
