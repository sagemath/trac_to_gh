```

    env SAGE_INSTALL_FETCH_ONLY=yes $(MAKE) -B SAGERUNTIME= \
        $(STANDARD_PACKAGES) gcc mpir python2
```
to use a new variable `$(SDIST_PACKAGES)` such that we can change `$(STANDARD_PACKAGES)` without affecting the sdist tarball.
