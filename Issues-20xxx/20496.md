This is incorrect docutils markup:

```
.. [2] E. Assmus, J. Key, Designs and their codes, CUP, 1992.
```
because the `E.` is interpreted as list item. It should be

```
.. [2] \E. Assmus, J. Key, Designs and their codes, CUP, 1992.
```
