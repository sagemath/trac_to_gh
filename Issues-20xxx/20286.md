```
sage: M = matrix(QQ, 2, 2, [1, 1/2, 1/3, 1/4])
sage: M
[  1 1/2]
[1/3 1/4]
sage: matrix(QQ, M.numpy())
[               1.0                0.5]
[0.3333333333333333               0.25]
```
The latter should give a matrix over QQ.
