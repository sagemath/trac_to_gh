It is good to differentiate between

```
sage: ZZ("x")
TypeError: unable to convert 'x' to an integer
```
and

```
sage: ZZ(x)
TypeError: unable to convert x to an integer
```

For this reason, consistently use `repr()` to format all such exceptions (similar to the `ZZ` example above).
