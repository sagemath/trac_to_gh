We currently have:

```
sage: C = crystals.Letters(['C',4])
sage: C(2) < C('E')
False
sage: C('E') < C(2)
# Boom
```
This is because we do not have a `_richcmp_` for `EmptyLetter`. This prevents the digraph code from drawing crystals with the empty letter (specifically KR tableaux).
