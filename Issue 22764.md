Elements of the following rings can't be checked for `NaN`:
* `CC` (`RR` can)
* `CDF` (`RDF` can)
* `CIF` (`RIF` can)
* `RLF`, `CLF`
* Ball fields

Other rings don't have `NaN` so there is no need for them.
