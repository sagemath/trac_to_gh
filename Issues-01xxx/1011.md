> Wouldn't that break `magma(25).is_zero()`?


You are right and thus I updated the patch:

```
sage: magma(9).is_zero()
False
sage: magma(0).is_zero()
True
sage: magma('false').bool()
False
sage: bool(magma(9).IsPrime())
False
sage: bool(magma(7).IsPrime())
True
```
