For example,

```
sage: X([3,2,1])*X([4,3,2,1])
X[6, 4, 2, 1, 3, 5, 7]
```

should be

```
sage: X([3,2,1])*X([4,3,2,1])
X[6, 4, 2, 1, 3, 5]
```
