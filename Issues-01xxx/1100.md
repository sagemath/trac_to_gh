According to the documentation, .roots() is only supposed to return values from the base ring, so this is a bug:

```
sage: x = polygen(ZZ)
sage: f = 2*x-3
sage: f.roots()
[(3/2, 1)]
```


Assignee: cwitty

Resolution: fixed
