CC:  @slel


```
a,b=var("a,b")
(a^2-2*I*a*b-b^2).factor()
((a-I*b)^2).expand()
```

outputs

```
a^2 - 2*I*a*b - b^2
a^2 - 2*I*a*b - b^2
```

So "factor" doesn't work correctly in this case.
