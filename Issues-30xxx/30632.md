```

```

```
----------------------------------------------------------------------

sage -t --random-seed=0 src/sage/databases/oeis.py  # 1 doctest failed
sage -t --random-seed=0 src/sage/misc/package.py  # 1 doctest failed
----------------------------------------------------------------------
```
