```

```

```
----------------------------------------------------------------------
sage -t --random-seed=0 src/sage/misc/remote_file.py  # 1 doctest failed

sage -t --random-seed=0 src/sage/misc/package.py  # 1 doctest failed
----------------------------------------------------------------------
```
