In sage-9.2.rc1: save? reports \\
 
"ValueError: allowed file extensions for images are '.eps', '.pdf', '.pgf', '.png', '.ps', '.sobj', '.svg'!" \\

Actually, it accepts '.html' just fine (at least for plots) :) 
Example:

```
save?
a = circle((0,0,0),1)
display(a)
save(a, 'save_test.html')
```
