For `__getitem__` in the case `n < -1`, would something like this make sense?

```diff
         if n < -1:
-            raise IndexError("index out of range")
+            try:
+                nn = len(self) + n + 1
+            except:
+                raise IndexError("index out of range")
+            if nn < 1:
+                raise IndexError("index out of range")
+            else:
+                return P.new("elt(%s,%s)" % (self._name, nn))
```
