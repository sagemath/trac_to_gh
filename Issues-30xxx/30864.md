Some timings.

Before the ticket:

```
sage: T.<y> = GF(5)[[]]                                                                                                   
sage: a = T.random_element(prec=1000)

809 µs ± 2.64 µs per loop (mean ± std. dev. of 7 runs, 1000 loops each)
sage: %timeit _ = ~a                                                                                                      
905 µs ± 3.22 µs per loop (mean ± std. dev. of 7 runs, 1000 loops each)
```

After the ticket:

```
sage: T.<y> = GF(5)[[]]                                                                                                   
sage: a = T.random_element(prec=1000)
```
