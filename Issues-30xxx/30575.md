If Maple is installed one can do:

```
sage: F = hypergeometric([-4, 2], [1], 1)
sage: G = maple(F); G
hypergeometric([-4, 2],[1],1)
```

From there, before this ticket:

```
sage: G.simplify()
hypergeometric([-4, 2],[1],1)
```

After this ticket:

```
sage: G.simplify()
0
```
