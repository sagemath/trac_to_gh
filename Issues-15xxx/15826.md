This looks wrong:

```
sage: R.<x>=LaurentPolynomialRing(QQ)
sage: f=(x^-1+x)/(1-2*x)
sage: f.divides(1/1)
True
sage: f.divides(1)
---------------------------------------------------------------------------
TypeError                                 Traceback (most recent call last)

----> 1 f.divides(Integer(1))
```
