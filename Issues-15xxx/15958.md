There is a bug in the implementation of the product in partition algebras.

```
sage: A = PartitionAlgebra(1,17)
sage: g = SetPartitionsAk(1).list()
sage: a = A[g[1]]
sage: a
P[{{-1}, {1}}]
sage: a*a
P[{{-1}, {1}}]
```

This should return

```
17*P[{{-1}, {1}}]
```
