Mark thinks atol() may be ok, though it could potentially introduce some limitations, which would in my opinion be undesirable.

```
long long my_atoll(char *instr)
{
  long long retval;
  int i;

  retval = 0;
  for (; *instr; instr++) {
    retval = 10*retval + (*instr - '0');
  }
  return retval;
}
```
