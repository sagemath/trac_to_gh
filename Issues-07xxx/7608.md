This is tricky, but not impossible to deal with. Usually, Python objects don't need to be told explicitly how to pickle themselves, but, for example, if you make a Cython object, you need to define a `__reduce__` method (see http://www.sagemath.org/doc/reference/sage/structure/unique_representation.html). Then we could probably just reach in and grab the adjacency dict from the old NetworkX object.
