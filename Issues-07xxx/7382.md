```

sage: K = Qp(3,5)
sage: J.<a> = K.extension(x^30-3)
sage: HJ = H.change_ring(J)
sage: import sage.schemes.hyperelliptic_curves.monsky_washnitzer as mw
sage: M_frob, forms = mw.matrix_of_frobenius_hyperelliptic(HJ)
```
