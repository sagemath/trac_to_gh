Ex: ` A[:,0:4:2]` should return the 0th and 2nd column of A, if possible:

```
age: A = matrix(ZZ,3,4, [3, 2, -5, 0, 1, -1, 1, -4, 1, 0, 1, -3]); A
[ 3  2 -5  0]
[ 1 -1  1 -4]
[ 1  0  1 -3]

sage: A[:,0:4:2]
[ 3  2 -5  0]
[ 1 -1  1 -4]
[ 1  0  1 -3]

```
