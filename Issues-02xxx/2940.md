The following should all work 

```
sage: var('x,y')
(x, y)
sage: (a < 1) + (b < 2)
b + a < 3
sage: (a < 1) + (b <= 2)
a + b < 3
sage: (a <= 1) + (b == 2)
a + b <= 3
```
