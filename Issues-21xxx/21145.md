<a id='comment:8'></a>
Calls like `self.__x.sort(cmp=_cmp)` will crash in python 3. I suggest you use `cmp_to_key()` from [functools](https://docs.python.org/2/library/functools.html):  
`self.__x.sort(key=cmp_to_key(_cmp))`
