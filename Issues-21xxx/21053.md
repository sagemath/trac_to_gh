This simple patch allows using `edge_color` as a default color together with `edge_colors`. To test try

```
g = graphs.PetersenGraph()
g.show(edge_color='red', edge_colors={'blue': [(1,2), (6,8)], 'green': [(5,8)]})
```

This is a part of #13827.
