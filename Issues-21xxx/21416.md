This package provides the GAP kernel for the Jupyter notebook,
making it available from the Sage's Jupyter notebook .

The source repository is at
- https://github.com/gap-packages/jupyter-kernel-gap

**Demo notebook**
- https://github.com/OpenDreamKit/OpenDreamKit/blob/master/WP4/D4.4/demo-gap.ipynb

**Tarball**
- https://pypi.python.org/packages/6d/02/2c88024c5851d8fc2a1340d1ffe9a2daf7d2fb0b72468e88348809d19ecd/jupyter-kernel-gap-0.9.tar.gz

Note. This is an optional package. To make it available to an existing Sage installation,
run the following command in a terminal:

```
sage -i gap_jupyter
```
