```
   1120         for i in range(1, n + 1):
   1121             for j in range(n + 1):
-> 1122                 mat[i, j] = c[m + i - j]
   1123         for j in range(n + 1):
   1124             mat[0, j] = z ** j

IndexError: list index out of range
```

Here is a quick fix:

```
for i in range(1, n + 1):
            for j in range(n + 1):
                if m + i - j < len(c): # new line
                # or
                # if m + i - j < self.degree(): 
                    mat[i, j] = c[m + i - j]
```
