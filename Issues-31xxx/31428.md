```
sage: A = AbelianGroupGap([])                                                                    
sage: A                                                                                          
Abelian group with gap, generator orders ()
```

Before this ticket:

```
sage: A([])                                                                                      
---------------------------------------------------------------------------
IndexError                                Traceback (most recent call last)
.....
--> 358             x = gens_gap[0]**0
    359             for i in range(len(exp)):
    360                 x *= gens_gap[i]**(exp[i] % orders[i])

IndexError: tuple index out of range
```

After this ticket:

```
sage: A([]) == A.one()
True
```

```diff
-            x = gens_gap[0]**0
+            x = self.one()
```
