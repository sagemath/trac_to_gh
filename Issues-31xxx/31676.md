```python
sage: from mpmath import mpc
sage: hash(mpc(1, -1))
---------------------------------------------------------------------------

<ipython-input-2-ff647837f542> in <module>
----> 1 hash(mpc(Integer(1), -Integer(1)))

   2496             True
   2497         """
-> 2498         return libmp.mpc_hash(self._mpc_)
   2499
   2500     def __neg__(s):

OverflowError: Python int too large to convert to C ssize_t
```
