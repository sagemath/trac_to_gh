```
sage: K = crystals.KirillovReshetikhin(['A',2,1], 2,1)
sage: view(K)

 DeprecationWarning: parameter {'backward':True} (in edge_options) is deprecated.
 Use {'dir':'back'} instead.

  return func(*args, **options)
```
