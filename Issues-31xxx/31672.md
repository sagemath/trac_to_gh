The following produces invalid Latex output (sage-9.2, misc seems not changed since):

```

sage: B.<x> = FreeAlgebra(A)
sage: (psi*x)._latex_()
'\\psix'
sage: repr_lincomb([(x, psi)], is_latex=True)
'\\psix'
```
