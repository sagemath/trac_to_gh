Needs input checking:

```
P.<x,y> = ProjectiveSpace(ZZ, 1)
H = Hom(P,P)
f = H([x^2-5*y^2, y^2])
D=f.dynatomic_polynomial(-3)
print D.parent(),D
F=f.dehomogenize(1)
D=F.dynatomic_polynomial(-3)
D.parent(),D
```

P.<x,y> = ProjectiveSpace(ZZ, 1)

f = H([x^2-5*y^2, y^2])
D=f.dynatomic_polynomial([1,0])
