Replace

```
TypeError: unsupported operand parent(s) for '+': ....
```
by

```
TypeError: unsupported operand parent(s) for +: ....
```
to be more consistent with Python:

```
>>> 1 + "foo"
TypeError: unsupported operand type(s) for +: 'int' and 'str'
```
