I suggest to give a hint, in the wiki/doc, that sage returns the characteristic polynomial in this way:

det(xI-A)

But there is also another definition:

det(A-xI)

which returns the same polynomial, but with different signs.

**Keywords:** characteristic polynomial, charpoly, linear algebra
