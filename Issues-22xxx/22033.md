Symbolic expressions support definite and indefinite integration:

```
sage: var('x')
x
sage: (x^3).integral(x)
1/4*x^4
sage: (x^3).integral(x, 0, 1)
1/4
```

Polynomials on the other hand support only indefinite integration:

```
sage: R.<x> = QQ[]
sage: (x^3).integral(x)
1/4*x^4
sage: (x^3).integral(x, 0, 1)
```
