We currently have

```sage
sage: F = FreeAlgebra(QQ, 3, 'x,y,z') 
sage: M.<x,y,z> = FreeMonoid(3)
sage: F.lie_polynomial(w)
x*y*z - y*z*x
```
and instead should result in:

```
x*y*z - x*z*y - y*z*x + z*y*x
```
