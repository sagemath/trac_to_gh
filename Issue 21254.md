```
$ sage -experimental
...
latte_int...............................1.7.3 (1.7.3)
...
```

```
sage: eqns = [[-10, 0, 1], [-10, 1, 0]]
sage: P = Polyhedron(ieqs=[], eqns=eqns)
sage: P.integral_points_count()
...
```
