Try this out (I tried both 8.2 and 9.1beta9):

```
sage: K.<z> = CyclotomicField(12)
sage: K1.<z1> = CyclotomicField(3)
sage: K(z1) in K1
True
sage: K1(K(z1))
z1
```
So far so good. But now:

```
sage: K(2) in K1
False
```
and correspondingly `K1(K(2))` breaks. 

More data points: 

```
sage: QQ(K(2)) in K1
True
sage: K(K1(2)) in K1
False
sage: K(2*z1) in K1
False
```
