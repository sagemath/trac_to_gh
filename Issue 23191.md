The latest and greatest OSX has some Applescript braindamage that prevents it from opening a browser. Workaround is 

```
export BROWSER=open
```

Links:
* Radar 31898264
* https://github.com/jupyter/notebook/issues/2438
* http://bugs.python.org/issue30392
