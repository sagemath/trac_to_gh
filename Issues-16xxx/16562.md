The error message `NotImplementedError` in case the ground
field is not perfect is a bit misleading: in general there will be no 
separable model. For instance,
    `k(t)(x,y| y^p = tx^p)`,
as a function field over k(t), doesn't have a separable model.

a separable model, or state that there is none. But this is not impemented yet.
