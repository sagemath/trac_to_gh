As the title says, we allow what seems quiet nonsensical:

```
sage: Set(None)
Set of elements of None
sage: Set(Set)
Set of elements of <function Set at 0x3e14230>
sage: Set(Ellipsis)
Set of elements of Ellipsis
```

This happens due to `sage.sets.set.Set_object`.
