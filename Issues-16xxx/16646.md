```
sage: type(Word(Permutations(range(3,52)).first()))

sage: type(Word(range(3,52)))
<class 'sage.combinat.words.word.FiniteWord_list'>
```

(NOTE: This ticket fix `set_partition_ordered` not `Words`!!)
