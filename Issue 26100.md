Current:

```
sage: h = e^-x
sage: latex(h)

```

Wish:

```
sage: h = e^-x
sage: latex(h)
e^{-x}
```

Likewise:

```
sage: a = exp(2*x)
sage: a  # good: we need those parentheses
e^(2*x)
sage: latex(a)  # should skip parentheses
e^{\left(2 \, x\right)}
```
and

```
sage: b = exp(x/2)
sage: b  # good: we need those parentheses
e^(1/2*x)
sage: latex(b)  # should skip parentheses
e^{\left(\frac{1}{2} \, x\right)}
```
