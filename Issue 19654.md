```
+    See [Hutz-gr]
```

```
`PGL(2,QQ)`
```

could be

```
`PGL(2,\QQ)`
```

```
-        lift,lifted = blift(LG,Li,p,S=S)
+        lift,lifted = blift(LG,Li,p,S = S)
```


`Abelian variety` should keep its capital A


```
+        returns ``True`` if this map is a morphism.
```

should be

```
+        Return ``True`` if this map is a morphism.
```


`greens function` should be `Green's function`

`+        THis map must`

`+        This map must`
