Consider the following (in sage 4.3.5):

```
sage: integral(exp(-x^2), x, 17, 42)   
-1/2*sqrt(pi)*erf(17) + 1/2*sqrt(pi)*erf(42)
sage: N(-1/2*sqrt(pi)*erf(17) + 1/2*sqrt(pi)*erf(42))
0.000000000000000
```
