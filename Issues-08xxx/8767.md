```
### libpng-1.2.35.p0 (Jaap Spies, Feb 1th, 2010)
 *

### libpng-1.2.35 (Michael Abshoff, April 10th, 2009)
 * update to latest upstream due to security issue in libpng (#5696)

```

The hg log ends

```
changeset:   13:ae01944f408c
```
