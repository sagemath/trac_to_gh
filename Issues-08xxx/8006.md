See #6729

This fails silently

```
CELL 1:
 u = 2
  u = 3
CELL 2:
print u # = 2
```

This should fail with an `IdentationError`, as it does in `%python`:

```
CELL 1:
 u = 2
  u = 3
# generates IdentationError
```
