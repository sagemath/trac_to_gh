We all know binomial(n,0) should be 1.  But we're not getting that answer in the following case.

```
sage: var('n, k')
(n, k)
sage: binomial(n, 0)  # this is OK
1
sage: binomial(n, k).subs(k=0)  # this is a problem!
0
```
