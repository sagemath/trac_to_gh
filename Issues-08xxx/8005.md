```

sage: R.<x,y>=F[]
sage: I=Ideal(R, [x, y])
sage: S.<x1,y1>=QuotientRing(R,I)
sage: print x1^2
x1^2
sage: print x1^3
x1^3
sage: print (x1^2)^2
x1^4
sage: print x1^4
NotImplementedError
```
