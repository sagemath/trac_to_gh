The formatted printing line below produces an unexpected output.

```
print(f"""
a: {1:0.2f}
a: 2
a: 3
""")
```

Instead of 

```
```
a: 1.00
a: 2
a: 3
```
```

It produces

```
```
a: 1.00
a: _sage_const_2
a: 3
```
```

when running sage on a .sage file or

```
```
a: 1.00
a: Integer(2)
a: 3
```
```
when run in interactive Sage, instead of the desired

```
```
a: 1.00
a: 2
a: 3
```
```

**Status:** new
