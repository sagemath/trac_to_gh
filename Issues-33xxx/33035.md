```
....:     p = next_prime(ZZ.random_element(2**(bits//2 +1)),

....:     q = next_prime(ZZ.random_element(2**(bits//2 +1)),

we should be able to do

```
....:     p = q = ZZ.zero()
....:     while (p == q):
....:         p = next_prime(ZZ.random_element(2**(bits//2 +1)),

....:         q = next_prime(ZZ.random_element(2**(bits//2 +1)),
