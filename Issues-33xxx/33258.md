# "Launched png viewer for ..."
## example

```
sage: p = Graphics()
sage: for phi in xsrange(0, 2 * pi, 1 / pi):
....:     p += plot(sin(x + phi), (x, -7, 7), rgbcolor = hue(phi))
sage: p
Graphics object consisting of 20 graphics primitives

```

```
sage: sage: p = Graphics() 
....: sage: for phi in xsrange(0, 2 * pi, 1 / pi): 
....: ....:     p += plot(sin(x + phi), (x, -7, 7), rgbcolor = hue(phi)) 
....: sage: p 
....:
```
