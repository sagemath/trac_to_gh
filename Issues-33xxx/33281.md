```
sage: s,q,n,a = var("s,q, n, a")
sage: k_n = n*pi/a
sage: assume(q, "real")
sage: integrand = q * ((q^2+k_n^2))^((1-s)/2)
sage: integrate(integrand, q, algorithm="sympy")

```

