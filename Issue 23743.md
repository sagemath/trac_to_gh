You wrongly used Python commnents (starting with `#`) inside doctests. The following is allowed

```
    sage: # my comment
    sage: print(1)
    1
```

the following is better

```
My comment::

    sage: print(1)
    1
```

But the following is wrong

```
# my comment
sage: print(1)
1
```
