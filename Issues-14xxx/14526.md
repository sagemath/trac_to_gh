Assignee: jason, was

Currently, `implicit_plot` only accepts one function at a time; `implicit_plot([x == y^2, x == e^y], (x, -1, 1), (y, -1, 1))` returns an error. It should be able to accept a list of functions, as does `plot` and `polar_plot`.
