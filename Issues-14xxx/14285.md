If cartesian_product_iterator has a `__len__` method we can use it in `random.sample`.

We also fix a bug that yields an infinite loop:

```
sage: for a in cartesian_product_iterator([ZZ,[]]):
....:     print "hello world"
<INFINITE LOOP>
```
