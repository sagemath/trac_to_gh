<a id='comment:1'></a>Currently, `short_vector_list_up_to_length` returns a list `L` such that `L[i]` is a *list* of vectors of length `i`. Clearly, we can no longer do this with non-integral quadratic forms. What do you propose we do in that case? Add a flag to return simply a list instead of a list of lists?
