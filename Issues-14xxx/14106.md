```
sage: R = Zp(5,7)
sage: a = R(300)
sage: a
2*5^2 + 2*5^3 + O(5^9)
sage: a[:5]
4*5^2 + O(5^5)
```

This results from the following:

```
sage: a.slice(None, 5, 1) # correct
2*5^2 + 2*5^3 + O(5^5)
sage: a.slice(None, 5, None) # incorrect
4*5^2 + O(5^5)
```
