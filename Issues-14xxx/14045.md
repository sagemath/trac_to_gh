Currently infinity and float do not compare properly:

```
sage: oo > float('inf')
True
sage: oo == float('+inf')
False
```
because of

```
sage: InfinityRing(float('inf'))
A positive finite number
```

Sage-devel topic: http://groups.google.com/forum/?fromgroups=#!topic/sage-devel/GMv525C0YcY
