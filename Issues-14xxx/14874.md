```
sage: MS = MatrixSpace(GF(2),2,2)
sage: A = MS([1,0,1,1])
sage: A.is_mutable()
True
sage: map(lambda x: x.is_mutable(), A.rows())
[True, True]
sage: A.set_immutable()
sage: map(lambda x: x.is_mutable(), A.rows())
[True, True]
sage: A.rows()[0] = 5
sage: A.rows()
[(1, 0), (1, 1)]
sage: A.rows()[0].set_immutable()
sage: map(lambda x: x.is_mutable(), A.rows())
[False, True]
```
Same thing happens on `columns()` as well.

Apply [attachment: trac_14874-fix_matrix_row_col-ts.patch.gz](https://github.com/sagemath/sage/files/ticket14874/trac_14874-fix_matrix_row_col-ts.patch.gz)
