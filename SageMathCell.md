
 [SageMathCell](https://github.com/sagemath/sagecell) is an [open-source](https://github.com/sagemath/sagecell/blob/master/LICENSE.txt), scalable, and easy-to-use web interface to Sage, a comprehensive open-source math software system. Additionally, SageMathCell can be used to embed Sage computations into any webpage.

See [the SageMathCell website](https://sagecell.sagemath.org/static/about.html?v=66c8a2df011da3aea3dfa789fc6117f4) for more details, including how to embed it in your own web pages.
