# Issue 31520: Topological interior of manifold subsets

Issue created by migration from https://trac.sagemath.org/ticket/31757

Original creator: mkoeppe

Original creation time: 2021-04-30 05:02:11

We define a subclass of `ManifoldSubset` whose instances represents the topological interior of a given subset in the manifold.  It will be a superset of any open subset of the given subset.

Subsets provide a method `interior` to construct it.
