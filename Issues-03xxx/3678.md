Raising a multivariate polynomial to a fractional power is broken.
A TypeError should be raised, but it is rounded instead. 

```
sage: pr = PolynomialRing(QQ, "u,v")
sage: pr.injvar()
Defining u, v
sage: u^(1/2)
1
```
