Wow, here's an embarrassing bug in the cyclotomic linear algebra code:

```
sage: cf4 = CyclotomicField(4) ; z4 = cf4.0
sage: A = Matrix(cf4, 1, 2, [-z4, 1])
sage: A.echelon_form()

[1 0]
[0 1]
```

The attached patch fixes it.
