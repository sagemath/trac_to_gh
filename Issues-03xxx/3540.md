For example:

```
sage: M = Matrix(GF(2), 0, 0, 0)
sage: M
[]
sage: M.nrows()
0
sage: M.ncols()
0
sage: N = Matrix(GF(2), 0, 19, 0)
sage: N
[]
sage: N.nrows()
0
sage: N.ncols()
19
sage: W = M.augment(N)
sage: W
[]
sage: W.nrows()
0
sage: W.ncols()
0
```
