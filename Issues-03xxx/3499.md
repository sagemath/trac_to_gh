This fails:

```
sage: N1 = Matrix(CyclotomicField(6), 1, [1])
sage: cf6 = CyclotomicField(6) ; z6 = cf6.0
sage: N2 = Matrix(CyclotomicField(6), 1, 5, [0,1,z6,-z6,-z6+1])
sage: N1*N2
[         0          1      zeta6     -zeta6 -zeta6 + 1]
```

The attached patch fixes it.
