sage -t can check stuff in 

```
\begin{verbatim}
sage: 2 + 3
5
\end{verbatim}
```

I propose to also support 

```
\begin{lstlisting}
sage: 2 + 3
5
\end{lstlisting}
```
which looks so much nicer.
