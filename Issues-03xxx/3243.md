Wrap log2 in a function so it will work in cygwin.  In cygwin log2 is a macro:

```
sh-3.2$ grep log2 *.h
math.h:#define log2(x) (log (x) / M_LOG2_E)
math.h:#define log2f(x) (logf (x) / (float) M_LOG2_E)
```
