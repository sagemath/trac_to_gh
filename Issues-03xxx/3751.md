The output of the class_number() method for QuadraticFields is a Python integer. Would it be possible to change this so that it returned a Sage integer?

```
sage: K=QuadraticField(-1,'a')
sage: K.class_number()
1
sage: type(K.class_number())
<type 'int'>
```
