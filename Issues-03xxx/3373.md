For x a ZZ variable, 1/x claims to be an integer:

```
sage: var('x', ZZ)
x
sage: (1/x).sym_parent()
Integer Ring
```

But in Sage, integer division always gives a rational:

```
sage: parent(1/1)
Rational Field
```
