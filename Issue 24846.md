IPython 5.6 was released on 02 April 2018.

Among other improvements in this bugfix release, the
[IPython 5.6 release notes](http://ipython.readthedocs.io/en/stable/whatsnew/version5.html#ipython-5-6)
mention that

- IPython should start faster, as it no longer looks for all available
  pygments styles on startup ([PR #10859](https://github.com/ipython/ipython/pull/10859/)).
