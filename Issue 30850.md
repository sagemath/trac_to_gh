Before this ticket, e.g. in Sage 9.3.beta4:

```
sage: identity_matrix(ZZ, 2).is_sparse()
False
sage: diagonal_matrix(ZZ, 2, [1, 1]).is_sparse()
True
```


After this ticket, the sparseness should match.
