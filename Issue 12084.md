Assignee: @burcin

Somebody tried to do 

```
sum(binomial(3, k), k, 0, 3)
```

and got a weird error, but

```
sum(binomial(SR(3), k), k, 0, 3)
```

works.  This is because 3.binomial isn't sophisticated enough.
