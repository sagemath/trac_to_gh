```
	sage: a = float(1.0)
 	sage: QQ(a)
 	TypeError: Unable to coerce 1.0 (<type 'float'>) to Rational
```
Note that the following works:

```
 	sage: a = float(1.0)
 	sage: QQ(RR(a))
 	1
```

> Yes, this conversion is missing. It should be easy to implement.
> 
> - Robert
