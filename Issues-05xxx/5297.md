```
sage: v = vector({1: 1, 3: -2})  
sage: w = vector({3: 3})       
sage: v
(0, 1, 0, -2)
sage: w
(0, 0, 0, 3)
sage: v.pairwise_product(w)
(0, 1, 0, -6)
sage: v.dense_vector().pairwise_product(w)
(0, 0, 0, -6)
```
(The last line illustrates that dense vectors seem to work okay.)
