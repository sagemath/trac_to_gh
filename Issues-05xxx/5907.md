A constant polynomial `a` is incorrectly `_fast_float_`- evaluated as `a*x`:

```
sage: R.<t> = QQ[]
sage: f = t + 2 - t
sage: ff = f._fast_float_()
sage: ff(3)
6.0
sage: list(ff)
['load 0', 'push 2.0', 'mul']
```
