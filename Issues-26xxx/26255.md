<a id='comment:8'></a>Overall, LGTM, but the output of `coxeter_diagram` should be an immutable graph since it is cached. I would actually just build all of the data, then pass it in with the `immutable=True`, so

```python
        V = self.index_set()
        S = self.simple_reflections()
        E = []
        for i,j in combinations(V, 2):
            o = (S[i]*S[j]).order()
            if o >= 3:
                E.append((i,j,o))
        return Graph([V,E], format='vertices_and_edges', immutable=True)
```
