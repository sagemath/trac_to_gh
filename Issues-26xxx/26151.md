```
sage: R.<x> = QQ[]
sage: f = x^5 + x + 1
sage: X = HyperellipticCurve(f)
sage: J = Jacobian(X)

sage: K = KummerSurface(J)
sage: K = J.kummer_surface()
sage: phi = X.kummer_morphism()
```
