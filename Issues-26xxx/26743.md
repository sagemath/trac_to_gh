The `cyclotomic_part` function that I introduced in #20263 needs a tweak:

```
sage: P.<x> = PolynomialRing(QQ)
sage: f = x*(x-1)*(x+2)
sage: f.cyclotomic_part()
x^2-x
```
Namely, the factor of `x` should be removed.
