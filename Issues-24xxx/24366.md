The orientation of an undirected graph must have the same set of vertices. This is currently not the case. 

```
sage: G = Graph(1)
sage: next(G.orientations())
Digraph on 0 vertices
```
With this ticket, we get

```
sage: G = Graph(1)
sage: next(G.orientations())
Digraph on 1 vertex
```
