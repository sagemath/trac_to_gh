```
sage: from collections import Counter
```

This is ok:

```
sage: X = GeneralDiscreteDistribution([1,2,2^1023])
sage: Counter(X.get_random_element() for _ in range(100))
Counter({2: 100})
sage: Counter(X.get_random_element() for _ in range(100))
Counter({2: 100})
```

This is not ok:

```
sage: X = GeneralDiscreteDistribution([1,2,2^1024])
sage: Counter(X.get_random_element() for _ in range(100))
Counter({2: 66, 0: 34})
sage: Counter(X.get_random_element() for _ in range(100))
Counter({2: 67, 0: 33})
```
