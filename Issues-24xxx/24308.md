An annoying not implemented error

```
sage: kk.<a,b> = GF(2)[]
sage: k.<y,w> = kk.quo(a^2+a+1)
sage: K.<T> = k[] 
sage: (T*y)^21
Traceback (most recent call last):
...
NotImplementedError:
```

When the parent does not implement `.characteristic()` we simply fallback to generic powering.
