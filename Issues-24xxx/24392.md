Sometimes it is ok to refer to a variable via string, sometimes it's not. Depends on the kind of assumption.

```
var('a b')
assume('a', a > 1)
assume(a, "integer")
assume('b', b > 1)
assume('b', "integer")
assumptions()
```


```
(a, b)
[a > 1, a is integer, b > 1]
```
