<div id="comment:1" align="right">comment:1</div>

It's because the loop through the sequence is interrupted as soon as the partial GCD is one, which is only valid for integer GCD:

```
    for vi in v:
        g = vi.gcd(g, **kwargs)
        if g == one:
            return g
```
