This is not a harmless change:

```diff
-        return self.eval_helper(max_symbolic, builtin_max, None, args)
+        return self.eval_helper(max_symbolic, builtin_max, float('-inf'), args)
```

```
res = builtin_f(res, x)
```

Everything else is fine.
